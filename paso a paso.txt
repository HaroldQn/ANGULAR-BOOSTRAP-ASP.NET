1) Hacer la cadena de conexión en WebConfig:
	<connectionStrings>
		<add name="BDLocal" connectionString="Server=(local);Database=BD_CRUD;Trusted_Connection=True;TrustServerCertificate=True;" providerName="System.Data.SqlClient" />
	</connectionStrings>
2) Creamos en models(es una clase) los constructores
3) Creamos el Gestor
4) Creamos el Constructor
5) En Web Api Config
	GlobalConfiguration.Configuration.Formatters.JsonFormatter.MediaTypeMappings
    .Add(new System.Net.Http.Formatting.RequestHeaderMapping("Accept",
        "text/html",
        StringComparison.InvariantCultureIgnoreCase,
        true,
        "application/json"
    ));
	para convertir a json
6) También instalamos microsoft.AspNet.WebApi.Cors
7) en web config colocamos , config.EnableCors();
